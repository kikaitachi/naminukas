FAQ

What is the meaning of robot name "naminukas"?
"naminukas" is a Lithuanian word for a magical spirit which lives in peoples homes, guards and heals them. This matches well with the purpose of the robot: robot can act as mobile camera for security, can carry and use tools to perform useful tasks and could be extended with an additional sensor payload to monitor various parameters like presence of harmful gasses or radiation.
