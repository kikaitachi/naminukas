FAQ

What is the meaning of robot name "naminukas"?
"naminukas" is a Lithuanian word for a magical spirit which lives in peoples homes, guards and heals them. This matches well with the purpose of the robot: robot should be able act as mobile camera for security, could have a sensor payload to monitor various parameters like presence of harmful gasses or radiation and potentially perfom more useful task with addition of manipulator arm.
