About

Purpose of this project is to create a robot which can easily navigate inside buildings by walking on hard flat floor, walls or even ceiling. Here is a prototype in action:

For a small robot it is hard to climb stairs and next to impossible to reach high places like window sill without flying or climbing walls. Solution is to use vacuum suction grippers which allow to stick to hard flat surfaces like floor, walls and ceiling. Luckily most of human inhabited buildings consist of hard flat and orthogonal surfaces - ideal for vacuum suction grippers.

Motivation for such a robot is to have a versatile mobile platform for indoor automation. Potential applications could be:

  • Mobile camera which can be operated remotely. While cameras with pan and tilt mechanism are cheap and omnipresent they can never replace truly mobile platform which can perform close up inspection.
  • Mobile camera can be suplemented by various sensors for inspecting hazardous or hard to reach environments.
  • Adding manupulator or other tools would allow to perform maintenace tasks is high places where ladder or scaffolding are impractical.

Self-imposed design constrains are:

  • Weight should be about 2kg: 1.5kg for robot itself and 0.5kg for a payload.
  • Size should not exceed 30cm in any dimension, i.e. portable in hand luggage.
