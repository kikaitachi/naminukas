About

Purpose of this project is to create a portable robot which can easily navigate inside buildings. For a small robot it is hard to climb stairs and next to impossible to reach high places like window sill without flying. Small flying machine can't carry a significant payload and large flying machine is inherently unsafe to operate indoors. Solution is to use vacuum suction grippers which allow to stick to hard flat surfaces like floor, walls and ceiling. Here is a prototype in action:

While most of buildings consist of hard flat and orthogonal surfaces - ideal for vacuum suction grippers, soft and porous surfaces like a carpet are very common too. Solution is to use vacuum suction grippers as wheels and transform walking robot to two wheel balancing robot. Here is a prototype in action:

Dual mode of operation makes such robot a very versatile mobile platform for indoor automation. Potential applications could be:

  • Mobile camera which can be operated remotely. While cameras with pan and tilt mechanism are cheap and omnipresent they can never replace truly mobile platform which can perform close up inspection.
  • Mobile camera can be supplemented by various sensors for inspecting hazardous or hard to reach environments.
  • Adding manipulation or other tools would allow to perform maintenance tasks is high places where ladder or scaffolding are impractical.

Self-imposed design constrains are:

  • Weight should be about 2kg: 1.5kg for robot itself and 0.5kg for a payload. Currently prototype weights 1405g.
  • Size should not exceed 30cm in any dimension, i.e. portable in hand luggage.
