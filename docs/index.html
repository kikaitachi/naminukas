Goal

To create a robot which can navigate and reach most of places of a typical human building. For example robot must be able to navigate to a different floor and reach window sill.

Assumptions & constraints

  • Typical human building is made of mostly flat, hard and othogonal surfaces.
  • Building has atmosphere.
  • Robot would be "portable":
    • Have weight under 2kg
    • Don't exceed 30cm in any dimension

Prototype

In order to achieve goal with contraints above, I have chosen to make a pneumatic robot. Here is initial prototype in action:

Vacuum suction cups allows robot to "stick" not only to floot but also to wall and ceilings. Wall and ceilings are often less cluttered than floor so it might be easier to utilize them instead of climbing strairs.

FAQ

What is the meaning of robot name "naminukas"?
"naminukas" is a Lithuanian word for a magical spirit which lives in peoples homes, guards and heals them. This matches well with the purpose of the robot: robot should be able act as mobile camera for security, could have a sensor payload to monitor various parameters like presence of harmful gasses or radiation and potentially perfom more useful task with addition or manipulator arm.
Is project open source?
Yes, eventually. See donate section.
